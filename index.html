Connect HubSpot with Airbyte

Before connecting:

1. Make sure your Airbyte API key is valid by clicking "Test Airbyte Connection"

2. If the test fails, you need to generate a new API key:

  1. Go to Airbyte Cloud UI > Settings > Account > Applications
  2. Click "Create an application" and name it
  3. Copy the Client ID and Client Secret
  4. Update server.js with these credentials
  5. Generate an access token with these credentials

3. Once the connection test passes, you can connect your HubSpot account
